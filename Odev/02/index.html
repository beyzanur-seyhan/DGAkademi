CSS Combinators

Descendant Selector

Ata elementin içindeki belirtilen bütün çocuk elementleri seçer. Burada ata element önemlidir.

ATA ELEMENT (descendat-selector)

Çocuk elementler

İkinci p öğesi

Üçüncü p öğesi

Dördüncü p öğesi

Beşinci p öğesi

SECTION

Section içerisindeki bir p elementiyim. Ama hala maviyim. Çünkü ata kapsayıcım "descendat-selector"

Child Selector (>)

Sadece belirtilen ebeveyne ait olan çocuk elementleri seçer. Burada parent element önemli.

PARENT ELEMENT (child-selector)

İlk çocuk element (p)

İkinci çocuk elementi (p)

SECTION

Ben section içerisinde bir p elementiyim. Benim şu anki parent elementim section.

Adjacent Sibling Selector (+)

Kendisinden hemen sonra gelen ilk elementi seçer.

Ben ilk Elementim (.first-elem)
+ Ben div'den hemen sonra gelen span'ım

General Sibling Selector (~)

Kendisinden sonra gelen belirtilmiş bütün elementleri seçer.

.general-select ~ div*10
